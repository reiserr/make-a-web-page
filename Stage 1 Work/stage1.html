LESSON 1: THE BASICS OF THE WEB & HTML
HOW THE WEB WORKS
You use your computers browser to make an HTTP request via the internet. The internet makes the request to servers which respond with files (such as HTML Documents) and sends them back to the browser to be displayed on your screen.
HTML

HTML stands for Hyper Text Markup Link and is comprised of four main components:

1. Text Content: What you see
2. Markup: What it looks like
3. References to Other Documents: e.g. images and videos
4. Links to Other Pages

HTML is the basis for the majority of webpages.

TAGS & ELEMENTS

Tags and elements are part of HTML markup.

Tags alter the way certain content is displayed.

The term element refers to the tag and the content within the tag.

WHY COMPUTERS ARE STUPID
Computers are literal and non-intuitive. They rely solely on human instruction in order to function. If a human error is made the computer will not know how to properly behave until the error is rectified by a human.
INLINE VS. BLOCK ELEMENTS

HTML elements are considered either inline or block. Block elements form an invisible box of width and height around the content.

LESSON 2: CREATING A STRUCTURED DOCUMENT WITH HTML
A WEBSITE AS A HOUSE
If you compare a website to a house, HTML is the structure (where the walls are placed, etc), CSS is the style (what color are the walls, etc), and Java Script is the interactive components (TV remote, garage opener, etc.).
THE CODE TREE
The nodes of an HTML document are layed out as a tree-like structure called the DOM (document object model) or the DOM tree. Nodes (sideways triangles) are HTML elements each with an opening and closing tag.
BOXES EVERYWHERE

Everything you see on a webpage can be broken down into boxes.What you see within those boxes can be altered using different languages (HTML, CSS, Java, etc).

LESSON 3: ADDING CSS STYLE TO HTML STRUCTURE
WHAT IS CSS?

CSS stands for Cascading Style Sheets and is used to control the style of HTML elements. This is done by using the following steps:

1. Choose an HTML class or element to alter (this will be your selector).
2. Choose an attribute (color, font-size, etc.) of your selector that you would like to stylize.
3. Allocate a value (size, color, etc.) to your chosen attribute.

The combination of attribute:value is called a rule and will be applied to any part of your code indicated by the selector. Descending elements can also inherit CSS styling from their ancestor elements.

AVOIDING REPETITION
Repetitive CSS is difficult to maintain and confusing to look at. Avoid repetitive CSS by classifying like elements in your HTML markup with the same name. When you use the HTML class as your selector in CSS the style will be applied to every element within the class even though you only coded it once.
HOW TO INCLUDE CSS IN YOUR WEBPAGE
METHOD 1
Although best for smaller projects, you could write CSS within a style tag in the head element of your HTML.
METHOD 2
Create a .css file separate from your .html file and place both files in one folder. Link to your .css file using the link tag within the head tag of your HTML markup.
METHOD 3
Generally a poor method due to repetitiveness but it does occur sometimes so it's good to be familiar with it.

BOX SIZING

1. Each HTML element is a box and each box has four elements:

i) Contents
ii) Padding
iii) Border
iv) Margin

2. Due to the number of components in each box, sizing can be difficult.

3. There are two main techniques for box sizing:

i) Set box size in terms of percentage instead of pixels
ii) Set the value of the box-size attribute to border-box

4. Different browsers may display the same code differently. You should use the following prefixes infront of the box-size attribute:

i) -wbkit-
ii) -moz-
iii) -ms-
BOX POSITIONING

1. Because Divs are block elements (not inline) they automaticall take the width of the page.

2. Adding the rule display:flex (flexbox) to a Parent Div will override this behavior and allow the Children to appear next to each other.
